Web and HTML Basic Concepts

Lesson 1 - The Basics of the Web and HTML

HTML - the basis for all web pages

HTTP - the protocol used between the internet and servers to communicate


HTML uses markup tags. Tags include:
  • Name
  • b (bold)
  • em (emphasis) = italics

HTML elements can have attributes. Examples include linking to websites like this.

Images use the img tag. Use alt="text " for blind screenreaders. Img is an example of a "void " tag, i.e. it does not have a closing tag.

Inline tag examples include a, span, br, img, and strong.

Block tag examples include div, p, and form. Block elements form an invisible box around them.

Lesson 1.2 - Creating a Structured Document

Overview of the Concepts:

  • HTML is the structure of the page;
  • CSS is the style;
  • JavaScript provides any interactivity.

"DOM " is Document Object Mode - meaning tree-like structure of a page.

"CSS " = Cassading Style Sheets - Used to provide color, font, etc.

Language --> syntax + rules

When designing webpages, cut the pages into boxes. For box, use div tags to define them.

Text editors - downloaded Sublime, which automatically adds tags. Use tags to create the trees that create the pages. CSS defines the styles.

Lesson 1.3 - Adding CSS Style and HTML Structure

CSS adds style to the page. It reads directions from the HTML code to apply style rules to format the page.

Design of webpages are made up of boxes. Think about all the boxes on the page while designing the page.

Developer tools in the browser can provide assistance for designing.

Avoid repetition. Give the same style to similar elements so you can modify them all at once by changing a single line of CSS.
